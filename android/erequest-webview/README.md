# e-Request Android WebView

Native Android wrapper for the Engineering e-Request mobile app.

The main UI is served by the Laravel WebView/API server, while this Android project handles:
- WebView shell and navigation
- File upload/download integration
- Firebase Cloud Messaging registration and notification handling
- Notification channels and custom sounds
- APK update/download flow

Build outputs, Gradle caches, local SDK paths, and APK artifacts are intentionally excluded from Git.
